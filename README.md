# Calculator Application

A complete calculator web application with database storage and error handling.

## Features
- Basic arithmetic operations
- Calculation history
- Error handling
- Responsive design

## Installation
1. Copy to `C:\xampp\htdocs\calculator-app`
2. Start XAMPP (Apache & MySQL)
3. Import `sql/database.sql` via phpMyAdmin
4. Open `http://localhost/calculator-app/index.html`

## Technologies
- HTML5, CSS3, JavaScript
- PHP 7.4+
- MySQL

## Database Configuration
- Database: calculator_db
- Username: root
- Password: (empty)

## Support
For issues, check logs folder for error details.
